核心就是突显文件操作

在我们最初的前端开发中，我们使用 JS 操作的数据类型是什么？
  json 
  文件 图片 视频（内容）
  二进制
  前后端分离的项目，前端传给后端的不是二进制， 一般都是 Json 的格式 
  通过请求后端提供的接口，返回了一组数据，我们需要将它渲染在界面上
  var obj = {
    "name: 'zce'
    age: 40
  }

  我们操作是字符串 ， 二进制人家后端在操作

当我们有了 node 之后，就可以让 JS 运行在服务端，因此我们可以将它看做是一个开发语言了(python php java)
【当前上下文当中是将 Node.js当语言】

此时采用 JS 就可以操作 二进制了，因此就能完成常见的 ：文件IO ，网络IO
